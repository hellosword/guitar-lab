# AI 辅助交互设计方法

> 本目录存放 AI 时代的交互设计方法论及其在 Guitar Lab 项目中的具体运用。
> 方法本身跨项目通用，但这里的文档强调"如何落地"。

---

## 方法概述

AI 正在改变交互设计的工作方式：非专业人员无需掌握专业设计工具（如 Figma），仅凭自然语言描述即可快速完成从需求梳理到可交互原型的全流程。

本方法涵盖五大核心环节：
1. AI 对话式需求拆解
2. AI 生成用户旅程图
3. AI 生成交互流程图
4. AI 生成可点击原型
5. AI 辅助可用性测试

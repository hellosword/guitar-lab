# FreePats Clean Electric Guitar Samples

Samples in this directory are from:

- Sound bank: FSBS Electric Guitar Clean #1, small SFZ WAV edition
- Project: FreePats
- Source: https://freepats.zenvoid.org/ElectricGuitar/clean-electric-guitar.html
- Version: 2022-09-11
- License: Creative Commons CC0 1.0 public domain dedication

The app uses the included small SFZ WAV sample set and maps fretboard notes to
the nearest SFZ sample region at playback time.
